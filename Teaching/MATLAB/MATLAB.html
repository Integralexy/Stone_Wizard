Simple Discrete Models in Biology and MATLAB

This reading group was done with two undergraduates during Spring 2025 as part of the WDRP. The description of the project can be found here. This project was designed to introduce basic MATLAB coding and implementation for biological modeling. Since modeling using continuous differential equations is a commonplace topic for many applied math majors, I chose to focus on difference equations.

The main reference we used was:

"Mathematical Models in Biology: An Introduction" by Allman and Rhodes
